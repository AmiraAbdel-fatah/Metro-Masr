🚇 Metro Masr (مترو مصر)

Metro Masr is a high-performance Android application designed to provide a seamless commuting experience for Cairo's metro passengers.

Built using Jetpack Compose and following Clean Architecture principles, the app offers smart routing, real-time fare calculation, and a highly accessible UI.


Features

Smart Trip Planning: Automatically calculates the most efficient route between stations.

Fare Estimation: Instantly displays the ticket price based on the number of stations.

Journey Details: Provides a detailed timeline of your trip, including transfers (L1, L2, L3) and estimated time.

Recent Search History: Quick access to your most frequent trips.

Fully Bilingual: Seamlessly switch between Arabic and English with full RTL support.

Adaptive UI: Supports both Light Mode and a premium Dark Mode experience.

Modern UX: Features like auto-focus search, smooth animations, and a sleek Splash Screen.


Architecture & Tech Stack

The project is built with Clean Architecture to ensure scalability, testability, and separation of concerns.

Layers: * Data: Repositories, Local/Remote Data Sources.

Domain: Business Models and Business Logic.

Presentation: MVVM Pattern, ViewModels, and UI Components.

UI Framework: Jetpack Compose (100%)

Dependency Injection: (Mention if you used Hilt/Dagger, or just Manual DI)

Asynchronous Programming: Kotlin Coroutines & Flow.

Design System: Material 3.
